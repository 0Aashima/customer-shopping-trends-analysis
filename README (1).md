# 👩🏻‍💻 Customer Behavior Analysis Dashboard | Data Analytics Portfolio Project

This project demonstrates a complete end-to-end Data Analytics workflow using **Python, PostgreSQL, SQL, and Power BI**. The objective is to analyze customer shopping behavior, identify purchasing patterns, and build an interactive business dashboard for decision-making.

This portfolio project showcases industry-relevant skills including:

* 📊 Data Cleaning & Transformation using Python
* 🗄️ Database Management using PostgreSQL
* 💻 Business Analysis using SQL Queries
* 📈 Dashboard Development using Power BI
* 📑 Business Reporting & Data Storytelling

---

# 🎯 Project Objective

The purpose of this project is to analyze customer purchasing behavior and generate actionable business insights that help organizations:

* Understand customer buying patterns
* Analyze subscription behavior
* Identify high-revenue product categories
* Evaluate customer loyalty
* Study discount effectiveness
* Compare spending across customer segments
* Build interactive dashboards for business users
